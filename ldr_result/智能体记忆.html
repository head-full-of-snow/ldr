关键趋势

  • 层次化架构成为主流：H-MEM、HiMem、HiGMem、MemoryOS、HyperMem等大量工作表明，多层次（短期/中期/长期）层次化存储是解决长周期记忆的核心方向。
  • 图结构超越向量检索：GAM、Synapse、AriGraph、HyperMem等证明，图结构（普通图→超图）在捕捉复杂关联方面显著优于纯向量检索。
  • 从静态记忆到动态自我进化：ParamMem、Evo-Memory、GAM(DeepResearch)等工作表明，记忆不再是被动存储，而是支持智能体持续自我进化的核心能力。
  • 多智能体记忆共享：G-Memory等研究开启了多智能体间记忆共享与协作的新方向。
  • 标准化基准形成：LoCoMo、MemoryAgentBench、Evo-Memory三大基准已形成行业共识。
  • 工业落地加速：AWS AgentCore、Mem0、Letta等产品和服务推动记忆技术从研究走向生产。
  • 认知科学启发：从"AI meets Brain"到Experience Compression Spectrum，生物启发的记忆架构设计成为重要方向。

一、记忆架构设计（10项）

学术论文高价值

1. MEM1: Learning to Synergize Memory and Reasoning for Efficient Long-Horizon Agents

📅 ICLR 2026 Poster👥 Zhou Zhou et al.
端到端强化学习框架，使智能体在长周期多轮任务中以恒定内存运行。LLM推理生成的记忆整合所有先前上下文，实现近恒定上下文窗口。无需手动设计记忆管理机制。
创新点：推理驱动的记忆整合实现恒定内存；RL训练使智能体自主学会何时写入、如何压缩记忆。
arXiv链接
学术论文高价值

2. MemoryOS: Memory Operating System of AI Agent

📅 EMNLP 2025 Oral👥 BAI-LAB
受操作系统内存管理原则启发，设计三层层次化存储架构（短期记忆、中期记忆、长期个人记忆）和四个核心功能模块（存储、更新、检索、优化）。
创新点：层次化存储STM/MTM/LTM三级；动态更新与淘汰机制；类似OS的内存管理思想。
ACL AnthologyGitHub
学术论文高价值

3. HyperMem: Hypergraph Memory for Long-Term Conversations

📅 ACL 2026👥 J Yue, C Hu et al. (EverMind)
开创性的超图记忆架构，通过超边显式建模高阶关联（多个元素之间的联合依赖），解决传统RAG或普通图记忆将相关信息拆分过碎的问题。
创新点：三层超图记忆架构；超边建模高维关联（3+个元素间）；有效保留信息间的复杂关系。
arXiv链接
学术论文高价值

4. A-MEM: Agentic Memory for LLM Agents

📅 NeurIPS 2025 Poster👥 Wujiang Xu et al.
受Zettelkasten（卡片盒笔记法）启发，构建结构化、不断演化的记忆图谱，智能体驱动的主动记忆管理，而非被动存储。
创新点：受卡片盒笔记法启发构建结构化记忆图谱；智能体驱动主动记忆管理；动态图谱支持高效检索。

创新点：三层记忆范围架构；混合存储结合向量搜索与图数据库；自动去重和增量更新。

创新点：知识图谱引导的RAG同时利用场景和语义记忆；扩散激活机制实现记忆间关联检索。
