关键趋势

  • 层次化架构成为主流：H-MEM、HiMem、HiGMem、MemoryOS、HyperMem等大量工作表明，多层次（短期/中期/长期）层次化存储是解决长周期记忆的核心方向。
  • 图结构超越向量检索：GAM、Synapse、AriGraph、HyperMem等证明，图结构（普通图→超图）在捕捉复杂关联方面显著优于纯向量检索。
  • 从静态记忆到动态自我进化：ParamMem、Evo-Memory、GAM(DeepResearch)等工作表明，记忆不再是被动存储，而是支持智能体持续自我进化的核心能力。
  • 多智能体记忆共享：G-Memory等研究开启了多智能体间记忆共享与协作的新方向。
  • 标准化基准形成：LoCoMo、MemoryAgentBench、Evo-Memory三大基准已形成行业共识。
  • 工业落地加速：AWS AgentCore、Mem0、Letta等产品和服务推动记忆技术从研究走向生产。
  • 认知科学启发：从"AI meets Brain"到Experience Compression Spectrum，生物启发的记忆架构设计成为重要方向。

一、新型记忆架构设计

学术论文高价值

1. MEM1: Learning to Synergize Memory and Reasoning for Efficient Long-Horizon Agents

📅 ICLR 2026 Poster🆔 arXiv: 2506.15841👥 Zhou Zhou et al.
介绍了MEM1，一个端到端的强化学习框架，使智能体能够在长周期的多轮任务中以恒定内存运行。在每个回合中，LLM推理生成的记忆会整合所有先前的上下文，实现近恒定的上下文窗口。无需手动设计记忆管理机制。
💡 创新点：通过推理驱动的记忆整合实现恒定内存；RL训练使智能体自主学会何时写入、如何压缩记忆。

2. Mem0: Building Production-Ready AI Agents with Scalable Long-Term Memory

📅 ECAI 2025🆔 arXiv: 2504.19413👥 Prateek Chhikara et al.
介绍了Mem0，一个可扩展的记忆中心化架构，通过动态提取、整合和检索持续对话中的关键信息。实现了两个层级的记忆架构（用户记忆和智能体记忆），以及混合存储（向量搜索+图数据库）。
💡 创新点：三层记忆范围架构（用户/会话/智能体）；混合存储结合向量搜索与图数据库；自动去重和增量更新机制。

3. A-MEM: Agentic Memory for LLM Agents

📅 NeurIPS 2025 Poster🆔 arXiv: 2502.12110👥 Wujiang Xu et al.

💡 创新点：受卡片盒笔记法启发构建结构化记忆图谱；智能体驱动主动记忆管理；动态图谱支持高效检索。

4. MemoryOS: Memory Operating System of AI Agent

📅 EMNLP 2025 Oral (长文)🆔 arXiv: 2506.06326👥 BAI-LAB

💡 创新点：层次化存储STM/MTM/LTM三级；动态更新与淘汰机制；类似OS的内存管理思想。

5. HyperMem: Hypergraph Memory for Long-Term Conversations

📅 ACL 2026🆔 arXiv: 2604.08256👥 J Yue, C Hu et al. (EverMind)
提出了开创性的超图记忆架构，通过超边显式建模高阶关联（即多个元素之间的联合依赖），解决传统RAG或普通图记忆将相关信息拆分过碎的问题。三层超图记忆架构，有效保留信息间的复杂关系。
💡 创新点：三层超图记忆架构；超边建模高维关联（3+个元素间）；有效保留信息间的复杂关系。

6. SYNAPSE: Empowering LLM Agents with Episodic-Semantic Memory via Spreading Activation

📅 2026-01🆔 arXiv: 2601.02744👥 Jiang, Chen et al.

💡 创新点：知识图谱引导的RAG同时利用场景和语义记忆；扩散激活机制实现记忆间关联检索；统一图结构建模两类记忆。

11. Experience Compression Spectrum: Unifying Memory, Skills, and Rules in LLM Agents

📅 2026-04🆔 arXiv: 2604.15877👥 Lewis Liu et al.
提出了经验压缩谱（Experience Compression Spectrum），统一框架将记忆、技能和规则定位为同一压缩轴上的不同点。利用互补学习系统（CLS）理论，将场景记忆"向上压缩"为新皮层知识，实现子线性可扩展的知识复用。
💡 创新点：统一框架：记忆→技能→规则的压缩谱；生物启发的"向上压缩"理论；实现子线性可扩展的知识复用。

12. Memory Management Impact Study: Empirical Study of Experience-Following Behavior

📅 2025-05🆔 arXiv: 2505.16067👥 多个研究者
对记忆管理策略（特别是添加和删除操作）如何影响LLM智能体行为和长期性能的实证研究。发现过度的记忆删除会损害智能体的经验遵循能力，而合理的记忆压缩能提升长期表现。
💡 创新点：首次系统研究记忆管理操作对长期表现的影响；发现记忆删除策略的潜在风险；提供记忆管理的实证指南。

15. RMM: Reflective Memory Management for Long-term Personalized Dialogue Agents

📅 ACL 2025🆔 arXiv: 2503.08026👥 谷歌研究员
提出反思性记忆管理（RMM）机制，整合了前瞻性和回顾性反思：前瞻性反思动态总结不同粒度的互动；回顾性反思从过去的对话中提取教训。在个性化对话场景中提升了长期记忆的一致性和相关性。
💡 创新点：前瞻性+回顾性双重反思机制；多粒度记忆总结；个性化对话优化。
